# Primeiros Projetos em Python 

Este repositório reúne meus primeiros projetos e exercícios desenvolvidos durante meus estudos em Python.

O objetivo é registrar minha evolução na programação, praticar lógica computacional e aplicar conceitos fundamentais da linguagem.

## Conteúdos estudados

* Variáveis e tipos de dados
* Entrada e saída de informações
* Operadores matemáticos
* Estruturas condicionais (`if`, `elif`, `else`)
* Estruturas de repetição (`for`, `while`)
* Funções
* Manipulação de arquivos
* Listas, tuplas e dicionários
* Programação orientada a objetos (em andamento)

## Mini-projetos

* Calculadora
* Conversor de temperatura
* Conversor de moedas
* Sistema de médias escolares
* Simulação de urna eletrônica
* Calculadora de tinta
* Outros exercícios de lógica e matemática

## Objetivos

* Fortalecer a base em Python;
* Desenvolver raciocínio lógico;
* Construir projetos cada vez mais completos;
* Evoluir para estudos em banco de dados, análise de dados e engenharia de dados.

## Próximos projetos
